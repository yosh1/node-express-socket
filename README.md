# node-express-socket
This is a chat application that developed by node.js / exapress and socket.io .

## How to run ?

### Local

```
$ npm start
```

### Docker

```
$ docker build -t chat .
$ docker run -it -p 3000:3000 chat
```

or

```
$ docker build -t chat .
$ docker run -it -p 3000:3000 chat ash
```

## How to deploy ? ( Now )
```
$ now projects add <project_name>
$ now --name <project_name>
$ now
```
